# 豆包 Pilot

让豆包网页版从「只会聊天」升级为「能动手干活」的浏览器扩展。

在 [豆包](https://www.doubao.com) 网页里，直接用自然语言让豆包帮你操作浏览器、查资料、填表单，甚至在你本机执行命令——全程在对话里完成，不需要切换工具。

## 能做什么

- **让豆包操作浏览器**：打开网页、阅读页面内容、点击按钮、填写表单，自动完成多步网页任务。
- **让豆包操作本机**（需安装本地助手）：执行命令行任务，把对话里的想法变成本机上的实际操作。
- **多步自动完成**：给一个目标，豆包会自己拆解步骤、逐步执行、根据结果继续，直到任务完成。
- **过程可见**：右下角面板实时显示每一步在做什么、结果如何，随时掌握进度。

## 设计目标

- **更易用**：装上即用，自动识别豆包页面，无需复杂配置。
- **更稳**：接口或页面有变化时尽量容错降级，不会一崩到底。
- **更强**：更清晰的工具协议与多步循环，让豆包把复杂任务真正做完。

## 安装

1. 下载/构建本扩展（开发版：`npm install && npm run build:chrome`，产物在 `.output/chrome-mv3`）。
2. 打开 `chrome://extensions`，开启「开发者模式」，点击「加载已解压的扩展程序」，选择构建产物目录。
3. 打开 [豆包](https://www.doubao.com)，正常对话即可。需要本机命令能力时，另行安装本地助手。

> 本项目为独立开源项目，仅适配豆包网页版。
